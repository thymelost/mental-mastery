# Section A — Validated Instrument Guides

These instruments are not reproduced here due to copyright. Each has a free publicly available version. Instructions for locating and administering each are provided.

---

## A1. TAS-20 — Toronto Alexithymia Scale

**What it measures:** Three dimensions of alexithymia — difficulty identifying feelings (DIF), difficulty describing feelings (DDF), and externally oriented thinking (EOT).

**Use in this framework:** Phase 0 baseline only. Determines entry track. Not re-administered.

**Where to find it:** Search "TAS-20 questionnaire Bagby Parker Taylor 1994" — freely available in academic paper appendices. Twenty items, 5-point Likert (1 = strongly disagree, 5 = strongly agree).

**Scoring:**
- Items 4, 5, 10, 18, 19 are reverse-scored (6 minus item score)
- Sum all 20 items for total score
- DIF subscale: items 1, 3, 6, 7, 9, 13, 14 (sum)
- DDF subscale: items 2, 4, 11, 12, 17 (sum)
- EOT subscale: items 5, 8, 10, 15, 16, 18, 19, 20 (sum)

**Framework interpretation:**

| Total Score | Framework Track |
|---|---|
| ≤ 51 | Standard — proceed normally |
| 52–60 | Borderline — extend Phases 1 and 2 by 2–4 weeks each |
| > 61 | Clinically significant alexithymia — somatic-priority track; extend Phases 1 and 2 by 4–8 weeks each; consider additional guided support for E1.1–E1.2 |

**Record:** Total score, three subscale scores, track assignment. Date.

---

## A2. FFMQ — Five Facet Mindfulness Questionnaire (Observing Subscale)

**What it measures:** The full FFMQ measures five facets: Observing, Describing, Acting with Awareness, Non-Judging, Non-Reactivity. **This framework uses the Observing subscale only**, as it is the only subscale that showed specific improvement in mindfulness vs. active control conditions (Yue et al., 2023).

**Use in this framework:** Phase 3 exit baseline, Phase 5 exit, L2→L3 gate, Phase 8 checkpoints. Tracks meta-attention (A2.1, A2.2) development.

**Where to find it:** Search "FFMQ Baer 2006 questionnaire" — freely available. The Observing subscale is 8 items (items 1, 6, 11, 15, 20, 26, 31, 36 in the standard 39-item version).

**Scoring:** Each item 1–5 (never/rarely to very often/always). Sum the 8 items. Range: 8–40.

**Framework interpretation:**

| Score Range | Interpretation |
|---|---|
| 8–20 | Low observing capacity — this is your Phase 3 baseline |
| 21–28 | Developing — typical range for Phase 3–4 practitioners |
| 29–35 | Functional — consistent with Phase 5 threshold |
| 36–40 | High — consistent with established dual presence capacity |

**What to track:** Direction of change across administrations, not absolute score. A 3+ point increase from Phase 3 to Phase 5/6 confirms meta-attention development.

## A3. SMS-Mind Subscale — Southampton Mindfulness Scale

**What it measures:** Mindful awareness of thinking and mental events — specifically the capacity to observe thoughts as events rather than as facts. Maps directly to C2.1 (Thought Observation) and C2.6 (Narrative Monitoring).

**Use in this framework:** Phase 3 exit baseline, Phase 4 exit, L2→L3 gate, Phase 8 checkpoints.

**Where to find it:** The SMS has two subscales: SMS-Mind and SMS-Body. Search "Southampton Mindfulness Scale Chadwick 2008" — freely available. SMS-Mind subscale is 8 items focused on mental events.

**Scoring:** Items 1–7 scale (not at all to completely). Sum subscale items. Higher = greater mindful awareness of mental events.

**Framework interpretation:** The absolute score matters less than change from baseline. A statistically meaningful increase from Phase 3 to Phase 4/6 confirms C2.1 and C2.6 are developing. If SMS-Mind is not increasing by Phase 4, the thought observation practice (3B) is not landing — review practice quality.

**Note:** If SMS-Body subscale is NOT increasing alongside SMS-Mind, this is expected (consistent with Ganesan et al., 2025 findings) and indicates somatic skills require their dedicated practices rather than cognitive mindfulness exercises.

---

## A4. PHQ-9 — Patient Health Questionnaire

**What it measures:** Depression symptom severity over the past 2 weeks (9 items, 0–3 each, range 0–27).

**Use in this framework:** Track A emotional regulation proxy. Administered at Phase 2 exit (baseline), Phase 4 and 5 exit, L2→L3 gate. A decreasing trend is consistent with developing emotional literacy and metacognitive skills. An increasing trend is a safety flag.

**Where to find it:** Freely available from Pfizer/Spitzer — search "PHQ-9 questionnaire PDF."

**Scoring:** Sum all 9 items. 0–4 = minimal, 5–9 = mild, 10–14 = moderate, 15–19 = moderately severe, 20–27 = severe.

**Framework interpretation:**
- Decreasing score over Phases 1–5: expected and consistent with skill development
- Score increasing or not changing: examine whether the relevant practices are being done consistently; check DASS-21 for stress and anxiety components separately
- Score ≥ 15 at any point: pause intensive practice and assess — this level of depressive symptoms may require clinical support before continuing

## A5. DASS-21 — Depression Anxiety Stress Scales

**What it measures:** Three subscales: Depression, Anxiety, Stress (7 items each, 0–3, multiply subscale sum by 2 for standard DASS-42 equivalent score).

**Use in this framework:** Parallel to PHQ-9. Provides finer decomposition — useful for distinguishing whether distress is primarily depressive, anxious, or stress-related, which informs which practices to prioritize.

**Where to find it:** Freely available from the Psychology Foundation of Australia — search "DASS-21 questionnaire."

**Scoring:** Sum each 7-item subscale, multiply by 2. Compare to standard norms.

| Subscale | Normal | Mild | Moderate | Severe | Extremely Severe |
|---|---|---|---|---|---|
| Depression | 0–9 | 10–13 | 14–20 | 21–27 | 28+ |
| Anxiety | 0–7 | 8–9 | 10–14 | 15–19 | 20+ |
| Stress | 0–14 | 15–18 | 19–25 | 26–33 | 34+ |

**Framework interpretation:**
- High Anxiety (≥ 15): prioritize S1.4 (arousal detection) and M3.1 (voluntary relaxation)
- High Stress (≥ 26): check practice load and I1.4 quality — may be working too hard; check sleep
- High Depression (≥ 21): same clinical caution as PHQ-9 ≥ 15 — consider clinical support

---

## A6. Rumination Response Scale (RRS)

**What it measures:** Tendency toward ruminative, repetitive self-focused thinking in response to depressed mood (22 items, 4-point scale).

**Use in this framework:** Phase 4 baseline and exit, L2→L3 gate. Tracks C2.6 (Narrative Monitoring) development — should decrease as narrative catch practice develops.

**Where to find it:** Search "Rumination Response Scale Nolen-Hoeksema Morrow 1991" — freely available. Use the 22-item version.

**Scoring:** Sum all 22 items. Higher scores = more ruminative. No standard clinical cutoff — use as change score from baseline.

**Framework interpretation:** A decrease of 5+ points from Phase 4 baseline to Phase 4 exit is consistent with C2.6 developing. If score is not decreasing, narrative catch practice (4D) is either not being done consistently or is not reaching real-time detection.

## B1. Somatic Baseline Assessment

**Skills assessed:** S1.1 (Sensation Recognition), S1.2 (Interoceptive Accuracy — indirect)
**When:** Phase 0 baseline; Phase 1 exit
**Duration:** 15 minutes
**Materials:** Timer, journal

**Procedure:**
1. Lie on your back on a firm surface. Set a timer for 10 minutes.
2. Perform a slow body scan from head to feet. When you detect a sensation anywhere, say or write it.
3. For each sensation, record three things: (a) the body location, (b) the physical quality using only physical terms, (c) the intensity (1–5).
4. After 10 minutes, take 5 minutes to review and complete your record.

**Scoring:**
Count the following:
- Total distinct sensations detected (one per body region per session)
- Number described in physical terms (pressure, heat, tension, vibration, tingling, coolness, pulsing, heaviness, numbness, hollow) vs. emotional/interpretive terms (tense, nervous, sad, bad, uncomfortable)
- Number of body regions that were blank (no sensation detected)

| Score | Meaning |
|---|---|
| ≤ 3 distinct sensations | Very low baseline |
| 4–8 | Low-moderate |
| 9–15 | Moderate |
| 16+ | High |
| > 50% interpretive language | Body scan language training needed; not yet scoring S1.1 |

**Phase 1 exit threshold:** 5+ distinct sensations, ≥ 80% physical language, across 3 consecutive sessions in the same week.

## B2. Heartbeat Detection Task (HBD)

**Skills assessed:** S1.2 (Interoceptive Accuracy)
**When:** Phase 0 baseline; weekly through Phases 1–2; Phase 2 exit
**Duration:** 15 minutes
**Materials:** Timer, journal, method to measure actual pulse (finger on wrist or throat)

**Procedure:**

*Setup:* Sit quietly. Do not have caffeine within 90 minutes. Measure your resting HR first: count pulse at wrist for 60 seconds. This is your baseline HR for calculating expected counts.

*Task:*
Run 4 intervals in this exact order. Between each interval, rest for 30 seconds.

| Interval | Duration |
|---|---|
| 1 | 25 seconds |
| 2 | 35 seconds |
| 3 | 45 seconds |
| 4 | 25 seconds |

*For each interval:* When you say "start," count the number of heartbeats you feel internally without touching your body. Focus on the sensation of your heartbeat — chest, throat, or wherever you can feel it. When the interval ends, write your count immediately before doing anything else.

*After all four intervals:* Calculate expected counts:
- Expected count = (Resting HR ÷ 60) × interval duration in seconds

*Scoring:*
For each interval:
```
Accuracy_i = 1 - (|your count - expected count| ÷ expected count)
```
Mean accuracy = average of 4 interval accuracies.

If any accuracy is negative (your count was off by more than 100%), record it as 0 for that interval.

**Example:**
- Resting HR = 72 bpm
- Interval 1 (25s): expected = (72/60) × 25 = 30 beats. You counted 26.
- Accuracy_1 = 1 - (|26-30| ÷ 30) = 1 - 0.133 = 0.867 = 87%

> **If using an HR monitor (recommended when available):** Do not use the resting HR estimate as your reference. Instead, read the actual beat count recorded by the device for each interval and use that as your reference count. This removes estimation error — your HR fluctuates during the task, so a single pre-task resting measure is an imprecise baseline. The formula is identical; substitute actual device count for expected count. Record both your count and the device count for each interval. Note in your log that a monitor was used, as this produces a more accurate accuracy score and allows direct comparison across sessions.

**Interpretation:**

| Mean Accuracy | Level |
|---|---|
| < 50% | Very low interoceptive baseline |
| 50–64% | Low-moderate |
| 65–79% | Moderate — standard Phase 1 threshold met |
| 80%+ | High |

**Threshold:** ≥ 70% mean accuracy across 3 consecutive weekly administrations = S1.2 threshold met.

**Weekly log format:**

| Date | HR (bpm) | I1 (25s) Expected/Got/Acc% | I2 (35s) Expected/Got/Acc% | I3 (45s) Expected/Got/Acc% | I4 (25s) Expected/Got/Acc% | Mean Acc% |
|---|---|---|---|---|---|---|
| | | / / | / / | / / | / / | |

## B3. Arousal Calibration Crosscheck

**Skills assessed:** S1.4 (Arousal Level Detection)
**When:** Weekly through Phase 2; Phase 2 exit
**Duration:** 5 minutes
**Materials:** Timer, journal, pulse measurement

**Procedure:**
1. Before measuring your actual HR, estimate your current arousal: 0 (completely calm/flat) to 10 (maximally activated/tense).
2. Sit quietly for 60 seconds. Do not try to change your state.
3. Measure actual HR for 60 seconds.
4. Record the pair.

**Calibration:**
After 4+ weeks of weekly data, plot (or simply review) the relationship between your arousal estimates and your actual HR. For most people, a personal calibration curve will emerge:
- HR < 60: likely maps to Arousal 0–2
- HR 60–70: likely maps to Arousal 2–4
- HR 70–80: likely maps to Arousal 4–6
- HR 80–95: likely maps to Arousal 6–8
- HR > 95: likely maps to Arousal 8–10

These ranges vary substantially between individuals. Your personal curve is what matters.

**Threshold:** Self-estimates within ±2 arousal points of expected HR-based estimate in ≥ 70% of weekly checks over a 4-week period.

**Weekly log format:**

| Date | Subjective Arousal (0–10) | Actual HR | Expected Arousal (from personal curve) | Error (points off) |
|---|---|---|---|---|
| | | | | |

## B4. Emotional Vocabulary Audit

**Skills assessed:** E1.3 (Emotional Vocabulary Expansion), E1.1 (Primary Emotion Identification — indirect)
**When:** Phase 0 baseline; Phase 2 exit
**Duration:** 15 minutes
**Materials:** Timer, journal

**Procedure:**
1. Set timer for 10 minutes. Without any reference, write every distinct emotional state you can name. States must be genuinely different in feel/function — not mere synonyms.
2. After 10 minutes, count your list.
3. Score the list on three dimensions:

**Dimension 1 — Count:**
| Count | Level |
|---|---|
| < 10 | Low |
| 10–20 | Moderate |
| 20–35 | Developed |
| 35+ | Extensive |

**Dimension 2 — Granularity (score 1–5):**
- 1 = only basic categories (happy, sad, angry, scared, disgusted)
- 2 = slightly differentiated (happy/excited/content, sad/depressed/grief)
- 3 = meaningfully differentiated, covers positive and negative well
- 4 = fine-grained distinctions (dread vs. apprehension; shame vs. guilt; contempt vs. disgust)
- 5 = nuanced across valences, intensity levels, and social dimensions

**Dimension 3 — Balance:**
Count how many are positive-valence, negative-valence, neutral. Extreme imbalance (> 80% any one valence) flags a gap.

**Phase 2 exit threshold:** ≥ 20 distinct states, granularity ≥ 3.

## B5. Attention Sustaining Assessment

**Skills assessed:** A1.1 (Attention Sustaining), A1.3 (Attention Recovery — indirect)
**When:** Phase 0 baseline; Phase 1 exit
**Duration:** 12 minutes (10 min session + 2 min record)
**Materials:** Timer, journal, a physical counter or simple tally method during session

**Procedure:**
1. Set a timer for exactly 10 minutes.
2. Sit comfortably. Place attention on the physical sensation of breath at the nostrils or the chest rise/fall.
3. Each time your attention leaves the sensation object without you choosing to move it (you notice you've been thinking about something else), make a small mark — tap a finger, click a pen, make a tally in the margin. Return attention immediately.
4. When the timer ends, record: (a) total drift count, (b) longest estimated continuous stretch without drift, (c) what pulled attention most.

**Scoring:**

| Drift Count | Level |
|---|---|
| 20+ | Very low sustained attention |
| 11–19 | Low-moderate |
| 6–10 | Moderate |
| 0–5 | Consistent sustained attention |

| Longest Continuous Stretch | Level |
|---|---|
| < 1 minute | Very low |
| 1–3 minutes | Low-moderate |
| 4–7 minutes | Moderate |
| 8–10 minutes | High |

**Phase 1 exit threshold:** ≤ 3 drifts and ≥ 10-minute longest stretch, across 5 consecutive sessions within the same week.

**Session log format:**

| Session # | Date | Drift Count | Longest Stretch (min) | Main Distraction |
|---|---|---|---|---|
| | | | | |

## B5a. Attention Direction and Discrimination Protocol

**Skills assessed:** A1.2 (Attention Directing)
**When:** Phase 1 exit
**Duration:** 20 minutes (single session)
**Materials:** Printed cue list, timer, journal

**Procedure:**

**Part 1 — Direction (10 minutes):**
Using the cue list below, receive 10 direction cues spaced 60 seconds apart. For each cue, immediately redirect attention to the named target and time how long it takes to establish clear focus there. Record whether focus was established within 5 seconds (pass) or not (fail).

**Standard cue list:** left foot sole → right hand palm → chest center → belly → jaw → right shoulder → left knee → throat → forehead → lower back.

**Part 2 — Discrimination (10 minutes):**
Review the most recent 30-minute period of your experience before this assessment began. Identify as many attention movements as you can recall. For each one, categorize it as: *Chosen* (you decided to redirect attention) or *Pulled* (something captured attention without your intention). Record each in your journal with a one-sentence description.

**Scoring:**
- Direction: count cues completed with focus established within 5 seconds (target: ≥ 8 of 10)
- Discrimination: calculate % of recalled attention movements correctly categorized (verify against session log notes from that session)

**Phase 1 exit threshold:** ≥ 8 of 10 direction cues completed within 5 seconds; chosen vs. pulled discrimination accuracy > 75% verified across one full session.

**Session log format:**

| Cue # | Target | Focus ≤5 sec? | Cue Completed |
|---|---|---|---|
| 1 | Left foot sole | Y / N | |
| 2 | Right hand palm | Y / N | |
| ... | | | |

## B5b. Distraction Resistance Assessment

**Skills assessed:** A1.4 (Distraction Resistance)
**When:** Phase 2 exit
**Duration:** Three 15-minute sessions in distinct environments
**Materials:** Timer, journal, distraction environment (see below)

**Procedure:**
In each session, maintain focused attention on a single designated object (breath, a body region, or a neutral visual object) for 5 continuous minutes. The remaining 10 minutes are used for setup, logging, and environment transition.

**Distraction environments — use 3 distinct conditions:**
- **Auditory:** Mild ambient noise (coffee shop, public transit, TV on low in background)
- **Visual:** Busy or visually complex environment (cluttered workspace, outdoor setting with movement)
- **Internal:** Deliberately recall a mild pending concern or unresolved task at the start of the session, then maintain focus

**Scoring per session:**
- Count only *unrecovered* drifts: attention that left the object for > 30 seconds without self-initiated return
- Drifts caught and recovered within 30 seconds are *recovered drifts* — do not count against threshold

**Phase 2 exit threshold:** ≤ 2 unrecovered drifts per session, across ≥ 3 sessions in distinct distraction environments.

**Session log format:**

| Session | Environment | Unrecovered Drifts | Recovered Drifts | Pass? |
|---|---|---|---|---|
| 1 | Auditory | | | |
| 2 | Visual | | | |
| 3 | Internal | | | |

## B6. Meta-Attention Probe Assessment

**Skills assessed:** A2.1 (Meta-Attention)
**When:** Phase 3 exit
**Duration:** 1 week of structured probing
**Materials:** Phone for alarms, journal

**Procedure:**
Set 10 irregular alarms per day for 5 consecutive days (vary the times each day — do not use a fixed pattern). When each alarm sounds:
1. Immediately, before doing anything else, write: "Where is my attention right now?"
2. Rate the latency to awareness: (A) I knew immediately, (B) took 2–3 seconds to locate, (C) took 5+ seconds or was genuinely unclear.
3. Record the attentional object: what were you attending to?

**Scoring:**
- Accuracy rate: % of probes where you could identify an attentional object (vs. "genuinely unclear")
- Latency distribution: % of A, B, C across all probes

**Phase 3 exit threshold:**
- Accuracy ≥ 80% (can name an attentional object in 8 of 10 probes)
- Latency improving trend: more A responses in week 5 data than week 3 data

**5-day log format:**

| Day | Probe # | Time | Attentional Object | Latency (A/B/C) |
|---|---|---|---|---|
| | 1 | | | |
| | 2 | | | |
| ... | | | | |

## B7. Dual Presence Performance Test

**Skills assessed:** A2.2 (Dual Presence)
**When:** Phase 5 exit, L2→L3 gate
**Duration:** 35 minutes
**Materials:** A text to read (400–600 words of moderate complexity, new to you), timer, journal

**Procedure:**

*Part 1 — Rested baseline (15 minutes):*
Read the first half of the text normally. After reading, answer 5 comprehension questions you write yourself about the passage. Score: 0–5.

*Part 2 — Dual presence condition (15 minutes):*
Read the second half of the text. While reading, maintain concurrent background monitoring of your internal state. Every 3 minutes, an alarm sounds (set 5 alarms at 3-min intervals). When each alarm sounds, immediately write a 1-sentence internal state report: "Right now my emotional tone is ___, arousal is ___, attention quality is ___."

After completing the passage, answer 5 comprehension questions about the second half. Score: 0–5.

*5-minute review:*
Compare Part 1 and Part 2 comprehension scores. Review whether the internal state reports are specific (vs. generic) and whether they reflect genuine ongoing monitoring or rapid confabulation.

**Scoring:**

| Metric | Threshold |
|---|---|
| Part 2 comprehension / Part 1 comprehension | ≥ 85% ratio (e.g., if Part 1 = 5/5, Part 2 must be ≥ 4/5) |
| Internal state report specificity | ≥ 4 of 5 reports are specific (not "fine" or "normal") |
| State reports vary across the reading period | At least 2 distinct states noted across 5 probes |

**Note:** Prepare 5 different text pairs (Part 1 / Part 2) so each administration uses fresh material. Mark texts as used.

## B8. Load-Condition Assessment (I1.1)

**Skills assessed:** I1.1 (Skill-Under-Load)
**When:** Phase 5 exit (provisional), L2→L3 gate (required), Phase 8 12-week checkpoint
**Duration:** 30 minutes total across 3 conditions (10 min each)
**Materials:** Journal; access to conditions described below

**The three conditions must be tested on separate occasions (not the same day):**

**Condition A — Cognitive Load:**
Administer within 15 minutes of completing a cognitively demanding work task (writing, analysis, complex problem-solving). Do not rest first.

**Condition B — Emotional Activation:**
Administer within 30 minutes of a mild-moderate emotional event (a frustrating interaction, disappointing news, an anxious wait). Arousal should be S1.4 level 4–6 at time of testing.

**Condition C — Social Context:**
Administer within 15 minutes of completing a significant social interaction (meeting, conversation, group activity).

**For each condition, rate the following 6 skills on a 0–10 scale (0 = completely offline, 5 = partially functional, 10 = fully functional):**

| Skill | Rested Baseline | Cond A | Cond B | Cond C |
|---|---|---|---|---|
| A2.1 Meta-Attention | | | | |
| A2.2 Dual Presence | | | | |
| C2.1 Thought Observation | | | | |
| C2.6 Narrative Monitoring | | | | |
| B2.1 Self-Other Distinction | | | | |
| B2.3 Value Anchoring | | | | |
| **Mean** | | | | |

**Establish rested baseline first** (same rating scale, in a normal calm state).

**Threshold:**
- Provisional (Phase 5): mean score across all 6 skills ≥ 80% of rested baseline in ≥ 2 conditions
- Full (L2→L3 gate): mean score ≥ 80% of rested baseline in ALL 3 conditions
- Full (Phase 8): ≥ 80% in all 3 conditions sustained across 4 consecutive weeks of ecologically valid conditions

**Interpreting 80%:**
If rested baseline mean = 8.0, threshold under load = 8.0 × 0.80 = 6.4. Any mean below 6.4 in a condition = threshold not met for that condition.

## B9. Boundary Challenge Assessment

**Skills assessed:** B2.1 (Self-Other Signal Distinction), B2.2 (Incongruence Detection)
**When:** Phase 5 exit
**Duration:** 30 minutes per session × 3 sessions (across 3 weeks)
**Materials:** Chosen media content, journal

**Procedure:**
Three sessions using three scenario types from the curriculum (Practice 5D):
- Session 1: Media/persuasive content exposure
- Session 2: Social pressure scenario recall and reconstruction
- Session 3: Sustained perspective exposure

**After each session, complete this attribution record:**

```
Scenario type: _______________

Emotional state BEFORE: _______________ (name + intensity 0–10)
Emotional state AFTER: _______________ (name + intensity 0–10)

Attribution question 1 (B2.1):
Was the emotional shift internally generated (from my prior state, beliefs, or memories)
or externally introduced (from the content/person/environment)?
  □ Internal   □ External   □ Mixed

Evidence for this attribution: _______________

Attribution question 2 (B2.1):
Did any thoughts or opinions shift during the session?
  □ Yes   □ No   □ Uncertain

If yes: Does the shift feel like my own reasoning or like absorbed content?
  □ My reasoning   □ Absorbed   □ Unclear

Attribution question 3 (B2.2):
Did any of the shifted thoughts or emotions conflict with my Values Document?
  □ Yes   □ No

If yes: Did I detect the conflict in the moment?   □ Yes   □ No
```

**Threshold:**
- B2.1: internal vs. external attribution is made for all 3 sessions, with stated evidence. Attribution is not always the same answer (not systematically defaulting to "internal" or "external")
- B2.2: at least 1 values-conflict detected in real time across the 3 sessions

## B10. State Shifting Verification

**Skills assessed:** M3.2 (State Shifting)
**When:** Phase 7 exit
**Duration:** 30 minutes
**Materials:** Journal, timer

**Procedure:**
You must have defined 3 target states before this assessment (from Phase 7 curriculum). Each state should have a written description including somatic signature, cognitive mode, breath quality.

Run 3 timed trials:

**Each trial:**
1. Record your current state (name + somatic markers)
2. Attempt to shift to the designated target state. Start the timer.
3. When you believe you are in the target state, stop the timer and record the shift time.
4. Sustain for exactly 10 minutes.
5. After 10 minutes, self-rate state fidelity: 1 (not in target state), 3 (partially there), 5 (fully in target state)
6. Record the 3 somatic markers that confirmed you were in the target state.

| Trial | From State | Target State | Shift Time (min:sec) | Sustained 10 min? | Fidelity (1–5) | Somatic Confirmation |
|---|---|---|---|---|---|---|
| 1 | | | | | | |
| 2 | | | | | | |
| 3 | | | | | | |

**Threshold:**
- Shift time ≤ 5 minutes for all 3 trials
- All 3 trials sustained to 10 minutes
- Fidelity ≥ 4 for all 3 trials
- Somatic confirmation: 3 specific physical markers described for each (not "felt calmer" — specific: "breath rate slowed, jaw released, hands felt warmer")

## B11. Calibration Honesty Check (I1.3)

**Skills assessed:** I1.3 (Calibration Honesty)
**When:** Phase 3 exit (covers L1 skills), L2→L3 gate (covers all L1 + L2), Phase 8 checkpoints
**Duration:** 45–60 minutes
**Materials:** Journal, all previous phase exit data, this assessment

**Procedure:**
Complete the rating matrix below. For each skill:
1. **Self-rate** (1–5): your current ability level, independently, before reviewing data
2. **Evidence-rate**: review your actual logged data for this skill — what does the evidence support?
3. **Compute gap**: Evidence-rate minus Self-rate. Positive = underestimate; Negative = overestimate.

### Interpreting Results

**Calculate:**
- Mean gap (all assessed skills): positive = systematic underestimation, negative = systematic overestimation
- Standard deviation of gaps: high SD = inconsistent calibration

**Flag conditions:**

| Pattern | Interpretation | Action |
|---|---|---|
| Mean gap ≥ +1 (consistent underestimation) | Imposter syndrome pattern; skills may be better than believed | Note and proceed; re-examine at next check |
| Mean gap ≤ −1 (consistent overestimation) | **Critical flag** — protection skill is overrated | Do NOT advance phase; spend 2–4 weeks generating evidence for overrated skills |
| Any individual gap ≤ −2 (self-rate 2+ above evidence) | Specific skill being significantly overestimated | Target that skill specifically before gate |
| High SD with near-zero mean | Random — no systematic bias | Proceed if no individual gaps ≤ −2 |

**Record in journal:** Mean gap, direction, any flags, actions taken. Date.
